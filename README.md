# MiniCRM (Yarım Kalmış Proje)

Bu repo, küçük bir e-ticaret firmasının müşteri ve sipariş yönetimi için başlanmış ama tamamlanmamış bir MiniCRM API projesidir.

## Kurulum (Özet)
```bash
npm install
copy .env.example .env
npm run migrate
npm run dev
```
